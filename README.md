
# 🏛️ Suvidha Sahayak

### Smart Municipal Governance & Citizen Service Platform

Suvidha Sahayak is a full-stack municipal governance platform designed to streamline citizen service requests, complaint management, geo-fenced ward monitoring, document uploads, and AI-powered assistance.

Built for scalable, real-world government infrastructure supporting large populations and enterprise-level deployment.

---

* 🌐 **Frontend:** Deployed on Vercel
* 🛢 **Database:** MySQL (Hosted on Railway)
* ⚙ **Backend:** Node.js + Express (Deployed on Render)

## ✨ Key Features

### 🗺 Geo-Fenced Complaint System

* Ward-based boundary validation using GeoJSON
* Citizens can only register complaints within municipal zones
* Interactive Leaflet map integration

### 📝 Connection Request Management

* Submit new municipal service requests
* Track application status
* Role-based approvals

### 📂 Secure Document Upload

* Upload supporting documents
* Secure file handling via backend
* Linked to user requests

### 🤖 AI Assistant – “Suvidha Sahayak”

* Multilingual smart assistant
* Municipal query support
* Context-aware assistance

### 🔐 Secure Authentication

* JWT-based login system
* Role-based authorization
* Token validation middleware

### 📊 Modern Responsive UI

* Tailwind CSS styling
* Framer Motion animations
* Enterprise governance design
